# Bill Splitter

A group expense tracker that minimizes the number of transactions needed to settle shared debts.

## Features

- **Transaction Minimization** — Algorithm reduces the number of payments needed among a group
- **Expense Tracking** — Add and manage shared expenses in real time
- **Group Management** — Create groups, add members, and track who owes what
- **Settlement Summary** — Clear breakdown of who pays whom and how much

## Tech Stack

- **Frontend:** React.js
- **Backend:** Node.js, Express.js

## Getting Started

```bash
git clone https://github.com/iamadarsh05/bill-splitter
cd bill-splitter
npm install
npm run dev
```
